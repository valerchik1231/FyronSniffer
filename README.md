The plugin allows you to modify the items that the Sniffer digs up.
///
Плагин может изменить предметы, которые выкапывает Нюхач.

Plugin commands/Команды плагина:
/fs add (id) (chance) (material) (min) (max) - add a material / добавить материал
/fs add (id) (chance) (min) (max) - add the item in your hand / добавить предмет в руке
/fs list - show current loot / посмотреть текущий лут
/fs remove [id] - remove loot / удалить лут
/fs reload - reload the config / перезагрузить конфигурацию
